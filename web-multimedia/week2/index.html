    Site Architecture - First Steps

    Overview

    In this second lab assignment, you will accomplish several things:

    1. Set up the file/folder structure for your midterm website project, a site that will serve as a platform for submitting all lab homework going forward.

    2. Archive your first about-me.html webpage so that it is accessible at the following URL (where netID is your netiD): https://people.duke.edu/~netID/labs/01/index.html

    3. Develop two new webpages, using HTML5 sectioning tags and what you have learned about CSS layout and the Box Model:

      • A landing page that is accessible at the following URL (again, where netID is your netiD): https://people.duke.edu/~netID/

      • A "Labs" landing page will contain links to each of the labs completed over the course of the semester, accessible at the following URL: https://people.duke.edu/~netID/labs/

    Public_html Folder Structure Setup

    1. Find the folder on your local harddrive where you saved your first webpage, about-me.html.

    2. Within that directory, create folders with these precise names, all lower-case, no spaces:

      • css
      • images
      • js
      • labs
      • final-project

    3. Within the labs folder, create a sub-folder, named lab01.

    4. Move your about-me.html file into the lab01 subfolder.

    5. Rename the about-me.html file as index.html

    6. The directory structure should look like this:

    7. Now, "push" this directory structure to your public_html directory using FileZilla:

      1. Launch FileZilla.

      2. Clicking the ▼ arrow next to the Quickconnect button to access your saved connection settings for accessing via SFTP the server login.duke.edu on port 22 with your netID and password.

      3. Make sure you're in the public_html folder on the remote server (right-hand pane). You should see your original about-me.html file in the root of that directory.

      4. Find in the left-hand pane the local directory containing the file-folder structure you set up earlier.

      5. Click and drag the five new folders from the left pane to the right pane to upload them into the public_html folder.

    8. If all goes well, when you go to https://people.duke.edu/~netID/ you should see the directory struture you just created (e.g. five folders - css, final-project, images, js, labs -- and the original about-me.html page.)

    9. Visiting https://people.duke.edu/~netID/about-me.html should still display your first lab assignment.

    10. Visiting https://people.duke.edu/~netID/labs/lab01/ should display the archived version of your about-me.html page.
